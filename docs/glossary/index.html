Accession Assessment An activity with the goal of determining whether or not a specimen should be accessioned into a collection. This Document Collections
Accessioning The formal, documented process to legally add an object or group of objects to a museum collection. A single accession transaction occurs when one or more objects are acquired in the same manner, from one source, and at one time. Master Glossary for DOI Museum Property Directives Collections
Acquisition An activity with the objective of gaining physical custory of an object for a museum collection through purchase, donation, bequest, transfer, or fieldwork.
